## 1. 테스트 개요

### 1.1 테스트 목적

Spring Boot 기반 게시판 웹 애플리케이션의 주요 기능이 요구사항대로 정상 동작하는지 검증한다.
기능 테스트(로그인, 게시판 CRUD, 검색, 페이징)와 비기능 테스트(보안, 세션)를 수행하여 품질을 확인한다.

### 1.2 테스트 범위

| 구분 | 대상 | 비고 |
|---|---|---|
| 기능 테스트 | 로그인/로그아웃, 게시글 CRUD, 검색, 페이징 | 전체 기능 |
| 비기능 테스트 | SQL Injection, XSS, 세션 만료 | 보안 항목 |
| 제외 항목 | 부하 테스트, 성능 벤치마크 | 학기 범위 외 |

### 1.3 테스트 환경

| 항목 | 사양 |
|---|---|
| OS | Windows 11 / macOS Sonoma |
| 브라우저 | Chrome 최신, Edge 최신 |
| Java | JDK 21 (Eclipse Temurin) |
| Spring Boot | 3.5.11 |
| DB | MySQL 8.0 |
| 빌드 도구 | Gradle 8.x (Wrapper) |
| IDE | IntelliJ IDEA Ultimate |

### 결론

전체 51개 테스트 케이스 중 51건 모두 PASS 판정을 받았다.
발견된 결함 3건은 모두 수정 완료 후 재테스트를 통과하였다.
로그인/로그아웃, 게시글 CRUD, 검색, 페이징 기능이 정상 동작하며,
SQL Injection 및 XSS 보안 항목도 안전하게 처리됨을 확인하였다.
추가로 JUnit 5 기반 단위/통합 테스트 24건(PasswordUtil 6건, UserRepository 8건, GreetingService 5건, GreetingController 5건)이 모두 통과하였다.
